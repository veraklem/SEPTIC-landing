# InbiAccept — лендинг. Спецификация (автор: оркестратор)

Продукт: **InbiAccept** — биопрепарат для септиков и выгребных ям. Бренд-платформа: **INBIOLAB**.
Задача сайта: имиджевая + продающая страница, готовящая покупку на маркетплейсах.

## 0. Важные решения (не менять без оркестратора)

1. Сайт **светлый**. Никакого тёмного фона, dark-tech, космоса.
2. Язык — русский, живой, без канцелярита. Запрещены конструкции: «не только…, но и…», «как…, так и…», «позволяет повысить эффективность», «является современным решением», «комплексный подход», «уникальная формула», «инновационное решение».
3. Никаких юридически рискованных обещаний: не писать «полностью устраняет запах», «на 100%», «мгновенно». Всегда «помогает», «снижает при регулярном применении», «поддерживает».
4. Все данные, которых нет (дозировки, сроки, артикулы, ссылки, контакты), — placeholders в конфиге, помеченные комментарием `// TODO(client)`.
5. Брендинг в текстах сайта: **INBIOLAB / InbiAccept** (по брифу). На имеющемся рендере флакона написано «INBIOSTORE / ИНБИО-СЕПТ» — это временный placeholder-визуал, путь к картинке должен легко меняться через конфиг.
6. Tailwind CSS **v3.4** (не v4). React 18 + TypeScript + Vite. framer-motion, lucide-react, @fontsource-variable/montserrat (подключить subsets: cyrillic, latin).
7. Backend не делаем. «Получить консультацию» = mailto из конфига (placeholder). Формы заявки в v1 нет.

### Поверхности
- Контейнер: max-w-[1200px], px-5 (mobile) / px-8 (md+), mx-auto.
- Вертикальный ритм секций: py-20 md:py-28.
- Карточки: rounded-[28px].
- Light-glass рецепт (компонент SoftGlassCard): `bg-white/65 backdrop-blur-md border border-white/80 shadow-[0_8px_40px_rgba(14,103,127,0.08)]`. Hover: подъём `-translate-y-1` + тень чуть глубже, transition 300ms.

  - Secondary: bg-white, border border-aqua-100, текст ink, hover: border-aqua-300 + bg-aqua-50.
- Фоновые декорации: очень мягкие радиальные градиенты aqua-100/50 (blur-3xl), тонкая SVG-волна между секциями там, где уместно. Не перегружать.

### Анимации (framer-motion)
- Общий паттерн: fade-up при появлении в viewport (`whileInView`, `viewport={{ once: true, margin: '-80px' }}`), y: 24→0, opacity 0→1, duration 0.6, easing [0.22, 1, 0.36, 1]. Карточки в гриде — stagger 0.08.
- Hero: заголовок и элементы появляются каскадом при загрузке (не whileInView).
- ShinyText (компонент): для слова «InbiAccept» — текст с градиентом aqua-700→aqua-500→aqua-300, поверх — мягкий световой блик, проходящий раз в ~4с (background-position анимация линейного градиента с прозрачной белой полосой, background-clip: text). Никакого агрессивного блеска, всё тонко.
- Лёгкий parallax фоновой декорации hero (useScroll + useTransform, сдвиг до 40px).
- `useReducedMotion` — при prefers-reduced-motion отключать движение (оставлять opacity).

### Иконки
lucide-react, size 22–26, strokeWidth 1.8, цвет aqua-600, в круглой подложке bg-aqua-50 rounded-2xl p-3. Одна иконка на карточку, хаос запрещён.

### ProblemScenarios — «Когда нужен InbiAccept»
SectionLabel: «Ситуации». H2: «Когда нужен InbiAccept». Лид: «Септик — живая система. Ей нужна поддержка в моменты, когда привычный ритм сбивается.»
Грид 3×2 (md), карточки SoftGlassCard, фон секции foam. Иконки lucide подобрать по смыслу (Wind, RotateCcw, Users, Gauge, Thermometer, CalendarCheck — на усмотрение исполнителя из этого набора).

1. **Появился запах на участке** — Система напоминает о себе после перерыва или жары. Бактерии помогают переработать накопившуюся органику.
2. **Септик работает нестабильно** — После зимы или долгого простоя микрофлора «просыпается» медленно. InbiAccept помогает запустить переработку заново.
3. **Семья приехала на выходные** — Нагрузка на систему выросла в разы за один день. Микрофлоре нужна поддержка, чтобы справиться с объёмом.
4. **Яма быстро наполняется** — Органика разлагается медленнее, чем поступает. Бактерии поддерживают естественный процесс разложения.
5. **После жары запах сильнее** — Высокая температура меняет баланс в системе. Регулярное применение помогает вернуть его.
6. **Хочется просто не думать об этом** — Проще поддерживать систему по графику, чем разбираться с последствиями. InbiAccept — для регулярного ухода.

### Benefits — «Преимущества» (#benefits)
SectionLabel: «Кому подходит». H2: «Сделан под реальные сценарии». 4 карточки (2×2), каждая = ситуация + польза:

1. **Дача с нерегулярным проживанием** — Септик неделями стоит без нагрузки, а потом получает всё и сразу. InbiAccept помогает системе переживать такие перепады.
2. **Сезонная поддержка** — Летом воды больше: гости, душ, кухня. Регулярное внесение бактерий помогает системе держать темп.
3. **Выгребные ямы** — Там, где нет камер очистки, особенно важно поддерживать разложение органики и сдерживать запах.
4. **Понятное применение** — Не нужно разбираться в микробиологии: отмерили, вылили, повторили по графику. Всё.
